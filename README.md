# Portfolio Website

A clean, modern, developer portfolio built with semantic HTML, polished CSS, and lightweight JavaScript animations.  

This site serves as a personal landing page showcasing projects, experience, and contact information, with an emphasis on performance, minimalism, and user experience.

---

## Features

- Clean, professional layout inspired by modern product design
- Smooth scroll-triggered reveal animations using IntersectionObserver
- Minimalist Apple-style hero section
- Responsive grid-based project layout
- Hover transitions and subtle elevation effects
- Fully responsive design for mobile, tablet, and desktop
- Zero external libraries; fully implemented with vanilla HTML, CSS, and JavaScript

- `index.html` – Main portfolio structure containing the hero, projects, about, and contact sections  
- `css/style.css` – UI styling, animations, layout, and responsive design  
- `js/script.js` – Scroll-trigger animations and smooth navigation behavior
